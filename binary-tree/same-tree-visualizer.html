Is Same Tree?

Step-by-step trace of isSameTree — watch two trees traversed simultaneously, node-pair by node-pair, with short-circuit && stopping early on any mismatch.

Pattern
🎯 Pre-order DFS + Short-circuit &&
Simultaneously walk two structures. Check current node, then recurse into children using &&. First false = stop.
Same pattern unlocks: #572 Subtree · #226 Invert Tree · #101 Symmetric Tree
📋Problem Statement

Given two binary trees p and q, return true if they are exactly the same — same structure AND same node values at every position.

Both must be structurally identical (same shape) and value-identical (every corresponding node has same value).

Examples
[1,2,3] vs [1,2,3]true
[1,2] vs [1,null,2]false
[1,2,1] vs [1,1,2]false
Key Facts
Both null→ true
One null→ false
Same vals, diff structure→ false
💡Intuition + Hints
1
Handle null cases first. If both are null → same. If one null, one not → different. These are your two base cases.
2
After null checks, compare p.val === q.val. If values differ → return false. No need to recurse.
3
Use && chaining: p.val===q.val && isSame(p.left,q.left) && isSame(p.right,q.right). Short-circuits on first false.
4
No helper needed — the main function IS the recursive function. Elegant recursion.

Core insight: Two trees are the same if their roots match AND their left subtrees are the same AND their right subtrees are the same. It's pre-order because we check root before children. The && gives us free short-circuiting — the moment any pair doesn't match, recursion stops.

isSameTree.js
var isSameTree = function(p, q) {
  if (!p && !q) return true;    // ① both null → identical
  if (!p || !q) return false;   // ② one null → different structure

  return p.val === q.val              // ③ values must match
      && isSameTree(p.left,  q.left)   //    AND left subtrees match
      && isSameTree(p.right, q.right); //    AND right subtrees match
};

var isSameTree = function(p, q) {
  if (!p && !q) return true;   // ① both null → same
  if (!p || !q) return false;  // ② one null → diff
  return p.val === q.val
      && isSameTree(p.left, q.left)
      && isSameTree(p.right, q.right);
};
